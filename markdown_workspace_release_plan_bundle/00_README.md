# Markdown Workspace Release Plan Bundle

This bundle contains the synthesized AI Council review, new master v1 release plan, diagrams, checklists, grading tables, and agent prompt packs for a native macOS Markdown workspace.

## Contents

- `01_AI_Council_Review_Report.md` — extensive review and grades
- `02_Master_v1_Release_Plan.md` — synthesized master plan
- `03_v1_Feature_Scope.md` — must/should/defer scope table
- `04_Risk_Register.md` — implementation risk register
- `diagrams/` — Mermaid source diagrams and rendered PNG/SVG diagrams
- `checklists/Manual_QA_Checklist.md` — manual QA checklist
- `agent_prompts/Agent_Prompt_Pack.md` — copy-paste agent prompts
- `tables/AI_Council_Grades.csv` — grading matrix

## Recommended Usage

1. Start with `02_Master_v1_Release_Plan.md`.
2. Copy `agent_prompts/Agent_Prompt_Pack.md` into Cursor/Claude Code/Windsurf as needed.
3. Use `checklists/Manual_QA_Checklist.md` before every release candidate.
4. Keep `04_Risk_Register.md` visible while agents implement code.
